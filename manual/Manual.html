Manual de Usuario para la Aplicación de Spa

Introducción

Esta aplicación está diseñada específicamente para gestionar de manera eficiente las tareas relacionadas con clientes, pagos, reservas y tratamientos. Cuenta con una interfaz intuitiva y fácil de usar.

Marco Principal

El marco principal de la aplicación presenta las siguientes secciones:

1. Clientes:

Tabla de clientes: Muestra una lista completa de todos los clientes registrados en el spa.

Opciones:

  • Añadir cliente: Permite crear un nuevo registro de cliente, ingresando sus datos personales (id, dni, nombre, apellidos, email).
  • Buscar cliente: Facilita la búsqueda de un cliente específico por cualquier atributo.
  • Borrar cliente: Elimina un cliente seleccionado del registro.
  • Actualizar tabla: Actualiza la tabla para mostrar los cambios realizados en la base de datos.

2. Tratamientos:

Tabla de tratamientos: Muestra una lista de todos los tratamientos realizados en el spa.

Opciones:

  • Añadir tratamiento: Permite crear un nuevo tratamiento, ingresando sus datos (id, precio, duración, descripción, tipo).
  • Buscar tratamiento: Facilita la búsqueda de un tratamiento específico por cualquier atributo.
  • Borrar tratamiento: Elimina un tratamiento seleccionado del registro.
  • Actualizar tabla: Actualiza la tabla para mostrar los cambios realizados en la base de datos.

3. Pagos:

Tabla de pagos: Muestra un registro de todos los pagos realizados por los clientes.

Opciones:

  • Añadir pago: Permite registrar un nuevo pago, ingresando sus datos (id, fecha, cantidad, método de pago, id del Cliente).
  • Buscar pago: Facilita la búsqueda de un pago específico por cualquier atributo.
  • Borrar pago: Elimina un pago seleccionado del registro.
  • Actualizar tabla: Actualiza la tabla para mostrar los cambios realizados en la base de datos.

4. Reservas:

Tabla de reservas: Muestra una lista completa de todas las reservas realizadas por los clientes.

Opciones:

  • Añadir reserva: Permite crear una nueva reserva, ingresando al cliente, el tratamiento deseado y la fecha.
  • Buscar reserva: Facilita la búsqueda de una reserva específica por cualquier atributo.
  • Eliminar reserva: Elimina una reserva seleccionada del registro.
  • Actualizar tabla: Actualiza la tabla para mostrar los cambios realizados en la base de datos.

Beneficios para el Usuario

  • Optimización del tiempo: La aplicación automatiza tareas repetitivas, liberando tiempo para que el secretario se centre en la atención al cliente y otras responsabilidades importantes.
  • Mejora de la organización: La centralización de la información en un solo lugar facilita la organización y el seguimiento de los datos relacionados con clientes, pagos, reservas y tratamientos.
  • Reducción de errores: La automatización de procesos minimiza la posibilidad de errores humanos, garantizando la precisión de la información.
  • Mejora de la atención al cliente: Un acceso rápido y preciso a la información permite al secretario ofrecer una atención al cliente más eficiente y personalizada.
